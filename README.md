
## Car Rental Service
    
This is a web application to rent cars and also buy cars.
 

## Features
    
 - Sign up
 - Login
 - Choose car
 - Buy car
 - Order car
 


## Run Locally

Clone the project

```bash
  git clone https://github.com/Ikpia/car-rental-service
```

Go to the project directory

```bash
  cd car-rental-service
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
    node app.js

```
